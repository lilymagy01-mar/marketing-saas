# 🌸 LilyMag Global Marketing SaaS: SNS Master Conquest Guide (v4.0)

이 가이드는 전 세계 모든 꽃집 사장님들이 '마케팅 자동화 함대'를 소유할 수 있도록 제작된 **[초정밀 행동 지침서]**입니다. 각 플랫폼의 설정 화면을 눈앞에 둔 것처럼 안내해 드립니다.

---

## 1. 📂 기초 공사: SNS 비즈니스 계정 준비
모든 자동화 시스템은 **'비즈니스/크리에이터 계정'**에서 가장 강력하게 작동합니다.
* **설정 방법**: 각 SNS 앱 설정 -> [계정 유형] -> [비즈니스/프로페셔널 계정으로 전환] 클릭.

---

## 2. 🎥 동영상 플랫폼 (YouTube, TikTok, Douyin)

### [YouTube Shorts] - 전 세계 검색의 중심
1. **[연결하기]** 클릭 후 구글 계정 로그인.
2. 'LilyMag에서 다음 권한을 요청합니다' 팝업에서 **[YouTube 동영상 관리 및 수정]** 항목을 반드시 체크하세요.
3. **[계속]**을 눌러 완료합니다. 
4. ⚠️ **주의**: '유튜브 스튜디오' 설정에서 [아동용 채널 아님]으로 설정되어 있어야 자동 업로드가 원활합니다.

### [TikTok & Douyin] - 무조건적인 유입의 핵심
1. **[틱톡/도우인 연동]** 클릭.
2. QR 코드가 나오면 틱톡/도우인 앱의 스캔 기능으로 촬영합니다.
3. **[Authorize]** (승인) 버튼 클릭 시 나오는 권한 목록 중 **[Public Video Upload]**에 반드시 체크를 확인하십시오.
4. 확인 버튼을 누르면 연동 성공!

---

## 3. 📸 비주얼 커뮤니티 (Instagram, Facebook, WeChat)

### [Instagram Reels & Facebook] - 메타 통합 사령부
1. 인스타그램 계정과 **'페이스북 페이지(Page)'**를 반드시 연결해두어야 합니다. (인스타 설정 -> 비즈니스 센터 -> 페이지 연결)
2. **[Meta 연결]** 버튼 클릭.
3. 페이스북 로그인 후, **'LilyMag과 연결할 페이지'**를 선택하라고 나옵니다.
4. 내 매장의 인스타그램 이름이 적힌 페이지를 선택하고 **[다음]** -> **[완료]**를 누르십시오.

### [WeChat Moments - 微信朋友圈] - 중국 시장의 열쇠
1. **[WeChat Moment Tool]**을 클릭합니다.
2. 가이드에 따라 제공되는 **[전용 통신 브릿지]**를 PC나 스마트폰에 설치합니다.
3. AI가 생성한 '모멘트용 카드'를 **[원클릭 전송]** 버튼으로 내 위챗 친구들에게 즉시 노출합니다.

---

## 4. ✍️ 전문 블로그 플랫폼 (Naver, Google, WordPress)

### [Naver Blog] - 한국 꽃집 마케팅의 1순위
1. 네이버 아이디 로그인 후 **[내 블로그]** -> **[관리]** -> **[플러그인/연동 관리]** -> **[글쓰기 API 설정]**을 클릭합니다.
2. **[API 연결 URL]**과 **[아이디]**, **[암호(API 연결 암호)]**를 화면에 뜨는 대로 복사합니다.
3. LilyMag 대시보드 블로그 설정창에 **[복사한 값들]**을 하나씩 붙여넣고 **[저장]** 버튼을 누르면 끝!

### [Google Blogger / WordPress] - 글로벌 SEO의 심장
1. **[Google Auth]** 버튼 클릭.
2. 'Blogger 콘텐츠 관리' 권한에 **[허용]**을 누릅니다.
3. (워드프레스의 경우) 매장 홈페이지 주소와 **[App Password]**를 입력하라는 창이 뜹니다.
4. 워드프레스 관리자 화면 [사용자] -> [프로필]의 하단에서 생성한 암호를 복사해 붙여넣으면 연동 완료!

---

## 5. 🐦 마이크로 블로깅 (X - Twitter, Threads)

### [X (Twitter)] - 실시간 소식 전파
1. **[Twitter Developer Connect]** 버튼 클릭.
2. 본인 트위터 계정 로그인 후 뜨는 창에서 **[Authorize App]** (앱 승인)을 클릭합니다.
3. ⚠️ **중요**: 연동 후 트위터 설정에서 'App permissions'가 **[Read and Write]**로 되어있는지 확인하세요 (읽기 전용이면 글을 못 올립니다).

### [Threads] - 스타일리시한 SNS 소통
1. 인스타그램이 연결되어 있다면 자동으로 Threads 연동 권한이 활성화됩니다.
2. **[Threads Ignite]** 버튼을 눌러 인스타그램 권한을 한 번 더 확인하면 자동 업로드 준비 완료!

## 🛠️ 긴급 조치 가이드 (Troubleshooting)
* **Q. 사진이 안 올라가요**: 페이스북/인스타 비즈니스 설정에서 '콘텐츠 게시 권한'이 빠져있는지 확인하세요.
* **Q. 중국 플랫폼 연동이 끊겨요**: 중국 지역 특성상 IP 변동이 잦을 수 있으니, 주 1회 로그인을 다시 확인해주는 것이 좋습니다.
* **Q. 연동은 됐는데 글이 안 올라가요**: SNS 계정이 '비공개' 상태면 시스템이 접근할 수 없습니다. 반드시 **[공개 계정]**으로 돌려주세요.
